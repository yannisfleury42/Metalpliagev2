ASSETS REQUIS — Metal Pliage Website
==================================

Placez vos fichiers ici avec exactement ces noms de fichiers.
L'affichage des cartes produits s'adapte automatiquement.


── SECTION HERO (vidéo) ──────────────────────────────────────
Fichier : assets/video/hero.mp4
Format  : MP4 H.264, autoplay muted loop
Contenu : séquence tôle plate → machine pliage → couvertine finie
Durée recommandée : 10–30 secondes (loop sans coupure visible)
Résolution : 1920×1080 ou 3840×2160 (4K)


── SECTION PRODUITS (6 photos) ───────────────────────────────
Placez ces photos dans assets/images/

  tole-acier.jpg        — Tôle acier huilée (close-up surface)
  inox-304.jpg          — Feuille inox 304/316L (reflet satiné)
  inox-brosse.jpg       — Inox brossé (texture linéaire visible)
  couvertine.jpg        — Couvertine standard posée sur un muret
  couvertine-custom.jpg — Couvertine sur mesure (profil complexe)
  pliage.jpg            — Opération de pliage / presse-plieuse

Recommandations :
  - Ratio : 4:3 (ex : 1200×900 px minimum)
  - Fond sombre ou fond d'atelier industriel de préférence
  - Éviter les fonds blancs ou studio générique
  - Format JPG ou WebP, < 400 Ko par image


── SECTION FABRICATION (1 photo) ─────────────────────────────
  fabrication.jpg       — Vue d'atelier : presse en action, opérateur,
                          ou matière en cours de traitement
  Ratio : 1:1 à 4:3 recommandé
  Résolution : 1400×1050 px minimum


── NOTES ─────────────────────────────────────────────────────
- Les images manquantes affichent un placeholder orange automatiquement.
- Vous pouvez aussi utiliser des formats .webp ou .png.
  Dans ce cas, mettez à jour les chemins dans index.html.
- Pour un rendu premium, photos à éviter :
  banques d'images génériques, éclairage studio blanc, images clipart.
