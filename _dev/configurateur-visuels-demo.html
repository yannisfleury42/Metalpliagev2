Configurateur — Module Dimensions : 4 variantes visuelles

Toutes les variantes montrent les mêmes données : muret B=200 mm · retour A=40 mm · tôle C=250 mm · rejet R=10 mm · débord 25 mm chaque côté. Le muret reste modifiable dans toutes les variantes. Pick celle qui te plaît, ou un mix d'éléments — j'intègre ensuite dans le configurateur.

VARIANTE A

Style guide pose (Module 4)

Couvertine outline pointillé clair + muret avec joints + cotes oranges en brackets. Lecture immédiate.

  • Couvertine outline pointillé · suggère le profil sans bloquer le muret
  • Cotes orange en brackets · A (vertical), B (haut), C (au-dessus), débords 25 mm en orange clair
  • Muret texturé briques · joints clairs, ambiance chantier
  • R = 10 mm en hint discret à droite
VARIANTE B

Plan technique blueprint

Grille subtile, lignes fines bleues, cotes type plan d'atelier, tags arrondis pour les valeurs.

  • Grille blueprint en arrière-plan, ambiance plan d'atelier
  • Couvertine en bleu clair avec léger fill semi-transparent
  • Tags rectangulaires avec fond sombre pour chaque valeur
  • Flèches d'arrows sur les cotes (style technique pro)
VARIANTE C

Profil produit (couvertine pleine)

Couvertine remplie avec gradient alu réaliste sur muret béton clair. Cotes oranges minimales. Style commercial.

  • Couvertine pleine avec dégradé alu lumière → ombre, plus tangible
  • Muret béton avec ombrage propre, sans surcharge texturelle
  • Cotes en tags ronds avec fond noir, lecture rapide
  • Pas de stries dans la couvertine — finition lisse
VARIANTE D

Vue axonométrique (3D légère)

Couvertine en perspective douce pour visualiser l'enveloppement du muret. Profil + une face de profondeur.

  • Vue axonométrique légère : face avant + côté en perspective
  • Couvertine en 3 faces visibles (dessus, façade, côté) avec dégradés cohérents
  • Visualisation enveloppante : on voit comment la couvertine couvre le muret
  • Plus impactant visuellement, moins technique
À toi : choisis la variante préférée (ou un mix : ex "A pour les cotes + C pour la couvertine pleine"), je l'intègre dans configurateur.js → le SVG sera dynamique (les valeurs B, A, C, L des inputs mettront à jour le schéma en temps réel).
