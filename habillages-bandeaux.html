TYPE U

HABILLAGE
BANDEAU TYPE U

Le profil Type U enveloppe les trois faces d'un bandeau ou d'un bois de toiture : face avant, face supérieure, face arrière. Il assure une protection complète contre les infiltrations d'eau, les UV et les chocs mécaniques.

Fabriqué sur mesure selon vos cotes exactes, le profil Type U s'adapte à tous les formats de bois de charpente ou de contrecollé de façade. Le retour arrière (segment C) peut être différent du retour avant (segment A) pour s'adapter à l'épaisseur du mur.

  • Protection 3 faces — face avant, dessus, face arrière
  • Étanchéité totale aux infiltrations d'eau
  • Cotes sur mesure — segments A, B, C indépendants
  • Laquage époxy 80 microns — résistance UV garantie
  • Disponible en 19 coloris RAL standards + finitions spécifiques

TYPE L

HABILLAGE
BANDEAU TYPE L

Le profil Type L couvre deux faces d'un bandeau : la face frontale et le dessus. Plus simple à poser que le Type U, il convient aux configurations où la face arrière est protégée par la maçonnerie ou le bardage.

Son angle à 90° lui permet de s'insérer facilement en finition de toiture ou en protection de bord de dalle. Disponible dans tous les coloris RAL de notre gamme.

  • Protection 2 faces — face avant + dessus
  • Pose simplifiée — moins de contraintes de cotes
  • Idéal pour finitions de toiture et bords de dalle
  • Angle 90° standard — personnalisable via configurateur
  • Mêmes finitions RAL que la gamme complète
